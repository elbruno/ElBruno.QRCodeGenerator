# Image Generation Prompts for ElBruno.QRCodeGenerator

Use these prompts in your favorite image model (DALL·E, Midjourney, SDXL, etc.).

## 1) Blog Header Image (16:9)

### Prompt

Create a modern 16:9 hero image for a technical blog post about a .NET open-source QR code generator called **ElBruno.QRCodeGenerator**. Visual style: clean developer workspace, dark VS Code-like editor on screen, glowing QR code elements floating around, subtle C#/.NET vibe with blue and purple accent lighting, high contrast, polished and professional, cinematic but minimal. Include visual hints of multiple outputs: terminal ASCII QR, PNG/image QR, and payload icons (WiFi, email, URL, vCard). Composition should leave negative space on the left side for blog title text overlay. Ultra-sharp, editorial quality, no clutter, no watermark, no logos from other brands.

### Optional Negative Prompt

low quality, blurry text, distorted QR codes, messy composition, over-saturated colors, too many UI panels, meme style, watermark, unrelated brand logos

### Suggested Settings

- Aspect ratio: 16:9
- Style: photoreal + UI concept art hybrid
- Lighting: soft neon rim light

## 2) YouTube Thumbnail Image (16:9, Viral Style)

### Prompt

Design a high-impact 16:9 YouTube thumbnail image for a video titled conceptually: **"Build QR Codes in .NET FAST"** featuring the project **ElBruno.QRCodeGenerator**. Style: bold, energetic, viral tech thumbnail. Show a surprised/excited presenter silhouette on one side pointing to a giant glowing QR code. On the other side, show a stylized code editor with C# snippets and icons for ASCII, Image, and Payloads. Use dramatic contrast, bright neon cyan + magenta highlights, clean background separation, strong focal points, and dynamic motion streaks. Make it instantly readable at small size. Leave space for large text overlay areas but do not render readable text in-image. Crisp, modern, clickable, no watermark.

### Optional Negative Prompt

tiny unreadable details, dull colors, flat lighting, crowded layout, excessive text, realistic face artifacts, low contrast, blur, watermark, clickbait unrelated objects

### Suggested Settings

- Aspect ratio: 16:9
- Style: thumbnail-optimized, high saturation, high clarity
- Emphasis: strong subject separation and one dominant QR focal point

## 3) LinkedIn Post Visual (1:1)

### Prompt

Create a professional 1:1 LinkedIn post image announcing a .NET open-source project named **ElBruno.QRCodeGenerator**. Visual style: clean, modern, trustworthy, engineering-focused. Central glowing QR code with surrounding minimal iconography representing ASCII terminal output, image export, and payload generation (URL, WiFi, email, vCard). Include a subtle code editor background with C# syntax colors, but keep it elegant and not busy. Color palette: Microsoft-inspired blues with soft purple gradients, balanced contrast, polished corporate-tech look. Composition must be simple, premium, and scroll-stopping for a professional feed. No watermark, no stock-photo clichés, no unrelated logos.

### Optional Negative Prompt

cartoonish style, noisy background, too many elements, unreadable tiny UI details, low resolution, exaggerated clickbait aesthetics, watermark

### Suggested Settings

- Aspect ratio: 1:1
- Style: professional social media graphic
- Tone: premium, credible, modern
